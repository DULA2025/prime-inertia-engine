Analytic phase approach: Since Ξ(t) is real-valued, E(t) = Ξ(t)·exp(-iαt) has phase φ(t) = arg(Ξ(t)) − αt. Between consecutive Ξ-zeros, arg(Ξ) is constant (0 or π), so φ'(t) = −α < 0. At each zero γₙ, Ξ changes sign → arg jumps by −π → φ drops by π. Therefore φ(t) is analytically guaranteed strictly decreasing — no numerical unwrapping needed. Combined with |E(z̄)| < |E(z)| for Im(z) > 0 (which follows from the exp(αy) growth factor), both de Branges conditions hold → E(z) ∈ HB₂.
