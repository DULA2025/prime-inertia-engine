Cohn-Elkies → Weil Transfer — The Wall

Constructing the actual projection · Does 24D positivity survive to 1D? · Step-by-step computation

Step 1: The Cohn-Elkies auxiliary function for Λ₂₄. The CKMRV proof (2019) that Λ₂₄ is universally optimal uses a radial auxiliary function f(r) satisfying: (CE1) f(0) > 0, (CE2) f(r) ≤ 0 for r ≥ 2 (contact radius), (CE3) f̂(t) ≥ 0 for all t, (CE4) f(2) = f'(2) = 0 (double root). The spectral gap forces r₀ = 2 (no shorter vectors), and the optimizer has the form f(r) = p(r²) · e^(-πr²) where p is a polynomial tuned so f̂ ≥ 0. We model this with a degree-6 radial function that satisfies all four CE conditions.
Cohn-Elkies auxiliary function f(r) and its Fourier transform f̂(t)
Step 2: Radial restriction — 24D → 1D. The CE function f(r) is a radial function on ℝ²⁴. Its 1D restriction is the radial profile h(r) = f(r), but evaluated along a single radial direction. The Fourier transform changes: the 24D radial Fourier transform (Hankel transform of order 11) becomes a 1D Fourier transform after the dimensional reduction. Key fact: the Hankel-to-Fourier projection preserves positivity under specific conditions related to the Bessel function J₁₁(2πrt) being positive-definite. We compute h(x) = f(|x|) on the 1D slice and its 1D Fourier transform.

Step 3: Mellin reparametrization — additive → multiplicative. The Weil distribution acts on functions of log(n), not functions of n. The Mellin transform M[h](s) = ∫₀^∞ h(x) x^(s-1) dx converts the radial profile from additive (Fourier) to multiplicative (Dirichlet series) coordinates. We compute φ(t) = h(e^t) — the function in "log coordinates" — and its properties. This is where the CE double root at r = 2 becomes a constraint at t = log(2), which is directly related to the first term 196560 · 2^(-s) in the Leech L-function.

Step 4: Construct the even Schwartz test function g. The map Φ produces g(x) = φ(|x|) · ψ(x) where ψ is a smoothing window that makes g Schwartz-class. For g to be even: g(-x) = g(x), which is automatic since φ is composed with |x|. The autocorrelation (g⋆g*)(x) = ∫ g(t)g(t+x) dt is the test function that enters the Weil distribution. The key question: does the CE positivity (f̂ ≥ 0) survive through Φ to make W(g⋆g*) ≥ 0?

Step 5: Compute W(g⋆g*) — the moment of truth. The Weil distribution has three terms: W(g⋆g*) = (g⋆g*)(0)·log(π) − Σ Λ(n)/√n · (g⋆g*)(log n) + ∫ K(t)·(g⋆g*)(t) dt. We compute each term separately and check whether their sum is ≥ 0. The spectral decomposition W = Σ_ρ |ĝ(ρ)|² tells us W ≥ 0 if and only if the Mellin transform of g is concentrated on the critical line — which is exactly what the CE Fourier positivity (f̂ ≥ 0) should guarantee, IF the projection preserves it.
