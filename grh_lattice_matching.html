The GRH extension hypothesis: For each primitive Dirichlet character χ mod q, define α_χ = π/log(q). The HB function E_χ(z) = Ξ_χ(z)·exp(-iα_χ z) should be in the de Branges class, with the lattice whose kissing number is closest to q providing the geometric substrate. We compute this for all primitive characters with conductor q ≤ 100.

Modulation spectra for selected characters. For each character χ mod q, we compute cos(α_χ · γₙ) where γₙ are the first zeros of L(s,χ). Since exact zeros of general L(s,χ) require heavy computation, we use the Gram point approximation γₙ ≈ 2πn/log(q/(2π)) for the zero ordinates, which captures the correct density log(q)/(2π) and average spacing 2π/log(q).
